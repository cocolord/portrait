## config.ini

新建配置文件，内容如下

```
{
    "uid": 10001000,
    "dump_dir": "weibo/data/",
    "type": "weibo",
    "base_url": "http://m.weibo.cn/page/json",
    "cookie": ""
}
```

type 值及对应的 base_url

| type | base_url |
| ---- | -------- |
| weibo | http://m.weibo.cn/page/json |
