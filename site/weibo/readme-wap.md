## config.ini

新建配置文件，内容如下

```
{
    "uid": 10001000,
    "cookie": "",

    "dump_dir": "data",
    "type": "weibo-wap",
    "ua": "../../config/user-agent.json",
    "base_url": "http://weibo.cn/drwujun",
}
```

type 值及对应的 base_url

| type | base_url |
| ---- | -------- |
| weibo | http://m.weibo.cn/page/json |
| weibo-wap | http://weibo.cn/u/ |
